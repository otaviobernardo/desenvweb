O curso de Front-end ensina os fundamentos do desenvolvimento de interfaces de usuário para aplicações web. Os alunos aprenderão a criar layouts responsivos usando HTML, CSS e JavaScript, além de frameworks populares como React e Bootstrap. O foco é na estética e usabilidade, preparando os alunos para construir sites dinâmicos e interativos que proporcionem uma excelente experiência ao usuário.
